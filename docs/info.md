# DVD Screensaver VGA

## How it works

This project defines a DVD screensaver VGA animation for SystemVerilog. It makes use of a VGA timer and a small ROM to iterate over the VGA output pixels and display the DVD logo. When the DVD hits an edge, it changes colors and starts moving in the opposite direction.

## Files

vga_timer.sv comes directly from CSE 100 Summer 2024 with Ethan Sifferman. I wrote this file as part of a larger project.

The other Verilog files, dvd.sv and project.sv, were almost entirely AI generated by Claude. The testbench test_dvd.py was also AI generated.

## How to test

Run `make` in the test directory. This will run test_dvd.py. This test takes quite a while, as it is simulating multiple vga frames. Eight frames are simulated, and they will be displayed in the `test/vga_output` directory as pngs. You can view these pngs to ensure that the output frames will look correct.

Note: when running the test with Github Actions, I do not know where these .png images are stored, or if you can view them.

## External hardware

This project (should) interface with the TinyVGA module to output the animation. The vga outputs are on the 8 monodirectional output pins. There are no used inputs for this module.
